# Market Intel Radar Scope

## In Scope

- Market intelligence signal capture.
- Source document indexing.
- Topic clustering.
- Live knowledge topic development.
- Proposal input development.
- Offer design seeds.
- Publishing input capture.
- Capability development.
- Instruction and workflow management.

## Out of Scope

- Treating Google Drive as a passive archive.
- Using GitHub to store bulky or confidential client files.
- Allowing agents to freely reorganize Drive without explicit approval.
- Rewriting approved copy, brand rules, or client-facing structures without instruction.

## Main Output Types

- Live topic pages.
- Reusable insight notes.
- Proposal components.
- R&D seeds.
- Framework notes.
- Prompt packs.
- Quality gates.
- Session reports.
- Handover notes.
