# Session Start Protocol

Before starting a new work session:

1. Pull the latest version from GitHub.
2. Read `AGENTS.md`.
3. Read this file.
4. Read the relevant role file from `02_roles/`.
5. Check `07_locked_decisions/locked_decisions.md`.
6. Check the relevant workflow in `03_workflows/`.
7. State the task in one clear sentence.
8. Confirm whether the work touches GitHub instructions, Google Drive files, or both.

## Work Classification

### Instruction Work

Examples:

- Updating role cards.
- Updating workflows.
- Adding a quality gate.
- Adding a prompt pack.

Store in GitHub.

### Source/File Work

Examples:

- Sorting PDFs.
- Reviewing decks.
- Creating exports.
- Handling client files.

Store in Google Drive.

### Bridge Work

Examples:

- Creating an index of Drive files.
- Creating a source-to-topic map.
- Documenting folder contracts.

Store the instruction/index logic in GitHub. Keep source files in Drive.
