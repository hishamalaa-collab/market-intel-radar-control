# Control Repo Baseline

## Current Purpose

This repository is the instruction and control layer for Market Intel Radar / Knowledge Studio. It stores operating instructions, workflows, role cards, prompt packs, locked decisions, quality gates, and reusable templates.

## Confirmed Architecture

- GitHub = instruction/control layer.
- Google Drive = source documents/assets/project files.

## Current Readiness Status

This repo is a safe starter baseline, not yet a full operating system.

## Known Deferred Gaps

- Constitution migration.
- Role cards.
- Drive-to-topic templates.
- Quality gates.
- Locked decisions.
- Client language bank.

## Current Rule

Actual project changes should happen first. GitHub should be updated afterward to reflect confirmed decisions.

## Do-Not-Change List Until Actual Project Updates Are Complete

- Do not rewrite the operating architecture.
- Do not reorganize Google Drive.
- Do not replace starter workflows with speculative full systems.
- Do not expand role logic beyond confirmed project behavior.
- Do not migrate client language or locked decisions without approved source material.
- Do not treat placeholder files as final operating doctrine.
