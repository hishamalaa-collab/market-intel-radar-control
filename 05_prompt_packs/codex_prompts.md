# Codex Prompt Pack

## Review Repo Structure

```text
Read AGENTS.md and validate the repository against scripts/validate_structure.py. Report missing files, unclear instructions, and suggested improvements without modifying anything first.
```

## Add Workflow

```text
Create a new workflow file for [workflow name]. Follow the style of existing files in 03_workflows. Do not modify locked decisions. Update CHANGELOG.md with a concise entry.
```

## Refine Existing Instruction

```text
Review [file path]. Make the instruction clearer and more operational without changing the underlying decision. Preserve existing approved wording where possible. Show a diff-style summary of changes.
```

## Prepare Pull Request Summary

```text
Summarize the changes in this branch under: purpose, files changed, decisions affected, risks, and recommended review focus.
```
