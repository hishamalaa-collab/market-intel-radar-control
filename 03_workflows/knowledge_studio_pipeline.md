# Knowledge Studio Pipeline

## Flow

```text
Google Drive
→ Source Documents Index
→ Topic Clusters
→ Live Knowledge Topics
→ Topic Pages
→ Reusable Outputs
```

## Stage 1 — Source Capture

Capture files, reports, emails, links, downloads, and observations in the correct Drive location.

## Stage 2 — Source Indexing

Create or update a Source Documents Index with:

- Source title.
- Source type.
- Date captured.
- Source origin.
- Relevance.
- Possible topic links.
- Action required.

## Stage 3 — Topic Clustering

Group sources by emerging theme, commercial relevance, or capability relevance.

## Stage 4 — Live Topic Development

Turn clusters into live topic pages with:

- Core question.
- Signal summary.
- Why it matters.
- Useful sources.
- Reusable insights.
- Proposal implications.
- Publishing implications.
- R&D seeds.

## Stage 5 — Reusable Outputs

Convert topic pages into reusable artifacts:

- Proposal paragraphs.
- Sales notes.
- Training ideas.
- Frameworks.
- Slides.
- Client conversation prompts.
- Publishing seeds.
