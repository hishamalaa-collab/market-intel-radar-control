# Source-to-Topic Workflow

## Purpose

Convert source material into live knowledge topics.

## Steps

1. Select a source or source cluster.
2. Extract the central idea.
3. Identify recurring patterns.
4. Define the topic question.
5. Connect related sources.
6. Create or update a topic page.
7. Add reusable outputs.
8. Record proposal, publishing, and R&D implications.

## Quality Check

A topic is not ready unless it contains:

- Clear topic question.
- Source evidence.
- Practical implication.
- Reusable angle.
- Next action.
