# Google Drive Ingestion Workflow

## Purpose

Move from file accumulation to structured source intelligence.

## Rule

Do not sort only by file type. Sort by future use.

## Recommended Intake Questions

For each source:

1. What is this?
2. Where did it come from?
3. Why was it saved?
4. Which topic could it inform?
5. Is it actionable, reference-only, vendor landscape, archive, or unknown?
6. Is there a proposal, R&D, publishing, or capability implication?

## Output

Update the Source Documents Index and assign the source to a folder contract.
