# Proposal Workflow

## Purpose

Support proposal development using reusable knowledge, market intelligence, client context, and approved company standards.

## Inputs

- Client request.
- RFP or brief.
- Client context.
- Internal capability references.
- Relevant topic pages.
- Locked language or structure decisions.

## Process

1. Understand the client reality.
2. Identify explicit requirements.
3. Identify implied risks and liabilities.
4. Define solution logic.
5. Map scope and phases.
6. Align with Bbusinesss proposal standards.
7. Draft management discussion notes if needed.
8. Prepare client-facing structure.
9. Run proposal review checklist.

## Output Options

- Management analysis.
- Proposal outline.
- Compliance matrix.
- Scope warning.
- Final proposal content.
