# Proposal Review Checklist

## Before Drafting

- Client context reviewed.
- Request language mirrored.
- Explicit requirements identified.
- Implied risks identified.
- Scope boundaries stated.
- Delivery assumptions stated.

## Before Sending

- Executive summary aligns with client reality.
- Methodology is clear.
- Scope is not overpromised.
- Operational risks are visible.
- Commercial assumptions are visible.
- Bbusinesss standards are followed.
- Formatting is clean.
- Terminology is consistent.
