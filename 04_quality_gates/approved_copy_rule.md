# Approved Copy Rule

## Rule

Approved copy is locked unless the user explicitly asks for rewriting.

## Allowed Changes

- Typo correction when requested.
- Formatting without wording changes.
- Placement into a deck/document without rewriting.
- Translation only when requested.

## Forbidden Changes

- Reframing.
- Synonym substitution.
- Shortening.
- Expanding.
- Adding explanatory language.
- Combining with another note.

## Validation

Before finalizing, compare the output against the approved copy source.
