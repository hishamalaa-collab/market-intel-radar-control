# No-Reinvention Rule

## Rule

When copy, structure, visual direction, workflow, or positioning is approved, preserve it.

## Do Not

- Rewrite approved copy for style.
- Add new concepts without permission.
- Change the structure because it seems better.
- Replace approved terminology.
- Mix notes from separate tasks.
- Expand scope silently.

## Do

- Apply only the requested change.
- State what changed.
- State what was intentionally preserved.
- Flag conflicts before editing.

## Review Question

Did this output preserve the approved material and only change what was requested?
