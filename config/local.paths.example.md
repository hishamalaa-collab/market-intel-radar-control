# Local Paths Example

Copy this file and rename the copy to:

```text
config/local.paths.md
```

Then edit the paths for the current machine.

Do not commit `config/local.paths.md` to GitHub.

## Windows Example

```text
GOOGLE_DRIVE_ROOT=G:\My Drive
KNOWLEDGE_STUDIO=G:\My Drive\Knowledge Studio
MARKET_INTEL_RADAR=G:\My Drive\Market Intel Radar
EXPORTS=G:\My Drive\Market Intel Radar\Exports
```

## Mac Example

```text
GOOGLE_DRIVE_ROOT=/Users/<YourName>/Library/CloudStorage/GoogleDrive-<email>/My Drive
KNOWLEDGE_STUDIO=/Users/<YourName>/Library/CloudStorage/GoogleDrive-<email>/My Drive/Knowledge Studio
MARKET_INTEL_RADAR=/Users/<YourName>/Library/CloudStorage/GoogleDrive-<email>/My Drive/Market Intel Radar
EXPORTS=/Users/<YourName>/Library/CloudStorage/GoogleDrive-<email>/My Drive/Market Intel Radar/Exports
```

## Rule

Each machine can have a different path. That is why this file is local-only.
