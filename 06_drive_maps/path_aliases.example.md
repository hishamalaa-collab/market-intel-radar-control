# Path Aliases Example

Use path aliases in instructions so the system does not depend on one machine's local folder names.

```text
[DRIVE_ROOT] = local Google Drive root
[KS_ROOT] = [DRIVE_ROOT]/Knowledge Studio
[MIR_ROOT] = [DRIVE_ROOT]/Market Intel Radar
[KS_INBOX] = [KS_ROOT]/00_Studio Inbox/00_Unsorted
[KS_SOURCE_DOCS] = [KS_ROOT]/01_Source Documents
[KS_REPORTS] = [KS_ROOT]/02_Reports
[KS_MANIFESTS] = [KS_ROOT]/03_Manifests
[KS_SYSTEM] = [KS_ROOT]/99_System
```
